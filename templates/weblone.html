About The Dataset:

The Abalone Dataset was originally published at the UCI Machine Learning Repository, and can be found at Data Link. The original stated problem is estimate the age an abalone, which can be determined from the number of rings in their shell. However, counting the number of rings in an abalone shell is an expensive method. Thus, one possible solution is predict the number of rings of an abalone from characteristics like height, diameter, lenght and weight measurements.

The Abalone Dataset is composed of the following attributes:

Name Description
Gender M, F, and I (infant)
Length Longest Shell Measurement
Diameter Perpendicular to Length
Height With meat in shell
Whole weight Complete Abalone Weight
Shucked weight Weight of meat
Viscera Weight Gut Weight
Shell Weight After Being Dried
Rings Number of Rings

Here Lengths are defined in mm & Weights are in grams.

We can Measure the age by adding 1.5 to the number of rings obtained


Age of Abalone = (Rings+1.5) Years
